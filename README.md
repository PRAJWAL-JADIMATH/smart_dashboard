The News and Opportunity Tracking Dashboard is a real-time web application that collects global news using NewsAPI and presents it in a clean, modern dashboard interface. The system filters news into categories such as global updates, technology & AI, markets, and science, allowing users to easily navigate through relevant information. A key feature of the application is its ability to generate short, AI-based summaries of each news article using technologies like OpenAI, helping users quickly understand important updates without reading full articles. The interface, built using React, displays news in a card-based layout with headlines, images, and concise summaries, along with options to view the original source. The dashboard continuously updates to reflect the latest information, acting as a centralized platform for tracking global events and opportunities efficiently, without relying on external notifications or alerts.
